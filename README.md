ASTRANET: Hidden Sector - это ретро консольная RPG/roguelike вдохновлённая FTL, с фокусом на развитие персонажа и корабля. Действие разворачивается в далёком будущем, где вы - капитан разведывательного фрегата флота Солнечной Федерации.

✨ Что нового в этой версии (Alpha 1)

✅ Создание персонажа - выбор предыстории (8 классов), распределение характеристик, уникальные пассивные способности.

✅ Двухуровневая карта галактики - навигация между секторами (Альфа, Бета, Гамма, Омега, Зета) с процедурно генерируемыми связями.

✅ Интерактивная карта систем - системы отображаются в виде узлов с соединениями, переход только по существующим связям (как в FTL).

✅ Локальная карта системы 20×20 - исследуйте планеты (O), станции (H), аномалии (~), ресурсы ($) и врагов (Y). Каждая клетка имеет свой символ и цвет.

✅ Динамическая система видимости - туман войны, корабли (Y) скрываются вне радиуса обзора, статические объекты остаются в памяти.

✅ Полноценный инвентарь - сбор ресурсов, использование расходников (медпакеты, стимуляторы), учёт веса и перегруз, подтверждение при выбрасывании.

✅ Система диалогов на JSON - все диалоги хранятся в отдельных файлах, поддерживают условия и действия (проверка предметов, репутации, изменение репутации, запуск квестов).

✅ Квесты с прогрессом - журнал заданий (J), автоматическое отслеживание целей, условия и награды.

✅ Репутация и фракции - 11 фракций с уникальными отношениями, репутация влияет на диалоги и цены, отображается в отдельном окне (R).

✅ Торговля - станции имеют собственные запасы, цены зависят от репутации и типа товара, интерфейс покупки/продажи с выбором количества.

✅ Сохранение и загрузка - сериализация в JSON с метаданными (дата, версия, позиция), удаление сохранений. Полное восстановление мира (сектора, системы, локальные карты).

✅ Атмосферный визуальный стиль - псевдопараллакс на картах, плавное мерцание звёзд, цветовая дифференциация объектов.

✅ Управление - глобальные клавиши I (инвентарь), C (персонаж), B (корабль), J (квесты), R (репутация), ESC (меню паузы), Backspace (назад), F5 (быстрое сохранение).

⚠️ Известные проблемы

При очень быстром нажатии клавиш иногда может пропускаться ввод (из-за Thread.Sleep в главном цикле). Будет исправлено в следующих версиях.

Некоторые символы могут некорректно отображаться в старых версиях Windows Terminal. Рекомендуется использовать Windows Terminal или консоль с поддержкой UTF-8.

Иногда может некорректно работать загрузка сохранений из карт окружения сектора (матричная карта), так что аккуратнее!

🛠 Технологический стек

Язык: C# (.NET 8.0)

Интерфейс: консоль Windows (цвета, псевдографика)

Сериализация: Newtonsoft.Json

Архитектура: стейт-менеджер, модульная структура

▶️ Как запустить

Установите .NET 8.0 SDK.

Скачайте последний релиз или склонируйте репозиторий.

Откройте решение в Visual Studio (или используйте dotnet run из папки проекта).

При первом запуске в папке с игрой автоматически создадутся подпапки Data/Dialogues и Data/TradeGoods с примерами файлов.

Играйте!
